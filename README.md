# 🏠 Airbnb Data Analysis Project

## 📌 Project Overview

This project focuses on analyzing Airbnb dataset to uncover key insights related to pricing, room types, and customer behavior. Using Python libraries, we perform data cleaning, preprocessing, and visualization to better understand trends in Airbnb listings.

The goal is to extract meaningful insights that can help in decision-making for hosts, travelers, and business analysts.

---

## 🎯 Objectives

* Analyze distribution of room types
* Study price variations across listings
* Understand relationship between price and room type
* Examine trends in number of reviews over time
* Perform data cleaning and preprocessing

---

## 🛠️ Technologies Used

* Python 🐍
* Pandas
* NumPy
* Matplotlib
* Seaborn

---

## 📂 Dataset

The dataset contains Airbnb listing details such as:

* Price
* Room Type
* Service Fee
* Number of Reviews
* Last Review Date

---

## 🧹 Data Cleaning Steps

* Removed special characters (`$`, `,`) from price columns
* Converted columns into appropriate data types
* Handled missing and inconsistent values
* Converted date columns into datetime format

---

## 📊 Exploratory Data Analysis (EDA)

### 1. Room Type Distribution

* Entire home/apartment listings dominate
* Private rooms are second most common

### 2. Price Distribution

* Prices vary widely across listings
* Presence of outliers indicates premium listings

### 3. Price vs Room Type

* Entire homes are generally more expensive
* Private rooms are more affordable
* Significant overlap exists due to location and amenities

### 4. Reviews Over Time

* Reviews increased over time showing platform growth
* Sudden peaks indicate high demand periods
* Drop in later years suggests missing or limited data

---

## 📈 Key Insights

* Entire homes are the most preferred listing type
* Price is influenced by multiple factors, not just room type
* Airbnb usage has grown significantly over time
* Data cleaning is crucial for accurate analysis
